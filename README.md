## 什么是 DMIT 换 IP 机制？先建立基本认知

**DMIT 换 IP（IP Replacement）**，是指当你的 VPS 被分配的 IPv4 地址在中国大陆等地区遭到封锁，无法正常连接时，向 DMIT 提交工单申请更换一个新 IP 的操作。

关键点：DMIT 的换 IP 政策不是按机房分的，而是按**产品线**分的。Premium、Eyeball、Premium Secure、Tier 1——四条线各有各的规则，不能混着理解。

## Premium & Eyeball 产品线：大多数人用的这条

这是 DMIT 用户最集中的两条线，包括洛杉矶 LAX.Pro、LAX.EB，香港 HKG.Pro、HKG.EB，日本 TYO.Pro、TYO.EB，以及各种活动款（年付 36/39/49 美元那批）。

**正常情况下的免费换 IP 规则（来自官方 TOS）：**

- 没有购买 IP Care+ 的用户：每 **15 天**可以申请一次免费换 IP
- 购买了 IP Care+ 的用户：每 **7 天**可以申请一次免费换 IP

月付和非月付在这里没有区别，两者都是 15 天一次。

**但"免费"是有前提的。** 以下四个条件同时满足，才能申请免费换：

1. 购买时间超过 7 天（刚买的头 7 天之内，换 IP 需要付费）
2. 距离上次换 IP 超过 15 天
3. 服务剩余有效期超过 7 天（快到期的不行）
4. **IP 必须是被完全封锁**——ICMP 全部不可达

最后这条是最容易踩坑的。说直白一点：只要有任何一个国家还能 ping 通你的 IP，或者只是某几个端口被封（比如 443 被墙但 IP 本身能 ping），DMIT 就认为这个 IP "没有被完全封锁"，这种情况下换 IP 要收费。

**收费的情况：$5 一次，随时可换。**

不满足免费条件但想马上换的，5 美元提工单，当天处理。

## Premium Secure 产品线：换 IP 最贵

这条线对应洛杉矶 LAX.sPro 系列（带 5Tbps+ DDoS 防护的那几款）。

规则很简单，没有免费选项：

- 每次换 IP 收费 **$15**
- 两次换 IP 之间间隔至少 **30 天**
- 购买 IP Care+ 后：每次急换仍然收 $5

价格是普通套餐的 3 倍。考虑清楚再买，用这条线的一般是对 DDoS 防护有硬需求的业务场景，不是普通梯子用户的首选。

## Tier 1（Standard）产品线：IP 保障本身就不确定

Tier 1 这条线主要是国际线路，没有中国大陆优化，对应 LAX.T1、HKG.T1、TYO.T1 系列，走大带宽、价格相对便宜。

这条线的 IP 换法和前两条完全不同——核心问题不是"换 IP 收不收费"，而是**"DMIT 压根不保证给你的 IP 在全球都能用"**。

具体规则：

- **月付订单**：不保证 IP 可以在中国大陆等有网络审查的地区连通；如果 IP 一开始就不能用，DMIT 不负责免费换
- **季付及以上**：自带 IP Guarantee+ 保障，首次连接如果发现 IP 在敏感地区不可用，可以申请免费换一个
- **现有用户（已在跑的服务）**：不支持免费换 IP
- **付费换 IP**：$5 一次，每次间隔至少 7 天

想要 Tier 1 也有 IP 连通保障？购买时加购 **IP Guarantee+** 附加服务（月付适用，一次性付费），首次发现 IP 不可用可以申请换新。

讲真，如果你买 Tier 1 主要是为了跑梯子或者科学上网，这条线本身就不是最适合的选择。对 IP 连通性有刚需，还是去 Premium 或 Eyeball 系列。

## IP Care+ 到底值不值得买？

这个问题不是所有人都需要纠结。先问自己：**你多久会遇到一次 IP 被墙？**

如果你用 CN2 GIA 或 CMIN2 线路，IP 被完全封锁的频率其实不高——大部分时候遇到的是某个端口被识别、某个协议被干扰，但 IP 本身还活着。这种情况属于"ICMP 可达"，不满足免费换的条件，买不买 IP Care+ 都要付 $5。

所以 IP Care+ 真正有用的场景是：

- 你的 IP 确实频繁被完整封锁（测试 ICMP 也不通）
- 你不想等 15 天，想 7 天就能申请一次
- 你需要应急，不想等到下一个 15 天窗口

如果 IP 稳定性还行，偶尔被墙一次，15 天免费换一次已经够用，IP Care+ 就没什么必要加购。

## 换 IP 怎么操作？步骤说一遍

1. **先确认 IP 是否被完全封锁**：用在线工具测一下你的 VPS IP，看国内和国外的 ICMP（ping）和 TCP 各端口的连通情况。如果国外还能 ping 通，DMIT 会判定为"ICMP 可达"，收费换
2. **确认距离上次换 IP 超过 15 天**（Premium/Eyeball 线，没买 IP Care+）
3. **确认购买时间超过 7 天**，且服务剩余有效期超过 7 天
4. 满足以上条件后，**登录 DMIT 后台提交工单**，说明 VPS 服务 ID 和换 IP 原因
5. 等待客服处理，中文客服响应一般比较快，工单提交后通常当天能处理

不满足免费条件，工单里说明愿意付 $5 换，也是一样流程。

## 换 IP 没用怎么办？几个容易忽略的点

换了新 IP 还是连不上，先别急着再发工单。大概率是以下几种情况之一：

**新 IP 又被封了。** 换完之后如果继续用同样的协议、同样的端口配置，被再次识别封锁的概率不低。换 IP 之后通常建议换个端口、换个伪装方式，不然等于白换。

**不是 IP 问题，是协议被识别。** GFW 的封锁有时候针对的不是 IP，而是特征流量。这时候换 IP 没用，要换协议或者混淆方式。

**等待时间不够。** 新分配的 IP 上线需要一点时间，刚换完 IP 马上测，有时还没完全生效。

**还有一点很重要**：刚买的套餐，如果 IP 一开始就连不上，要**当天联系客服**，DMIT 对"首次连接不可达"有单独处理政策，超过了就按正常换 IP 规则走了。

## 常见问题

**Q：DMIT 换 IP 免费吗？**  
Premium 和 Eyeball 系列，IP 完全被封且满足时间条件（购买超 7 天、距上次换超 15 天、服务剩余超 7 天），免费换。不满足条件或需要立即换，$5 一次。

**Q：怎么判断我的 IP 是不是"完全被封"？**  
用在线 IP 检测工具，输入你的 VPS IP 和端口，看 ICMP 可达性。如果国内外都 ping 不通，才算完全封锁。只有某个端口被封或者部分连通，DMIT 会判定为"ICMP 可达"，收费。

**Q：月付套餐能免费换 IP 吗？**  
Premium 和 Eyeball 月付可以免费换，规则和季付相同（每 15 天一次）。Tier 1 月付不行，如果想有 IP 首次连通保障，需要加购 IP Guarantee+。

**Q：刚买就发现 IP 用不了怎么办？**  
购买当天就发现 IP 不可用，立刻联系 DMIT 客服工单说明情况，这属于"首次连接不可达"，Premium 和 Eyeball 系列保障首次连通，客服会处理。拖过了当天再找，就按正常换 IP 流程走。

**Q：IP Care+ 是什么，在哪里买？**  
IP Care+ 是换 IP 频率提升的附加服务，开通后每 7 天可免费换一次（默认 15 天）。在 DMIT 后台购买 VPS 或升级配置时可以选购。Tier 1 系列对应的是 IP Guarantee+ 而不是 IP Care+，两者是不同的附加服务。
